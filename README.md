## 🟡 주요 기능
- 회원가입, 로그인
- 카카오, 네이버 로그인
- 기억력 검사
- 집중력 검사
- 일기 작성, 수정, 삭제, 보기
- Chat GPT API 활용 기억력 및 집중력 향상 방법 제공

## 🟢 기술 스택
- React.js
- TypeScript
- React Router
- Zustand
- Axios
- Chart.js
- React Daum Postcode
- React Quill
- Day.js

## 🔵 프로젝트 실행 방법
### 클론 및 폴더 이동
```bash
git clone 깃주소
cd 프로젝트명
```

### 패키지 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run start
```

### 빌드 방법
```bash
npm run build
```
